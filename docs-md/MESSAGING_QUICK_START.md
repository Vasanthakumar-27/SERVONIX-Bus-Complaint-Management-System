## Γ¥ô FAQ

**Q: Can I send messages to multiple people?**  
A: No, messages are only between one Admin and Head.

**Q: Can I edit a sent message?**  
A: No, messages cannot be edited after sending.

**Q: What happens if I link a deleted complaint?**  
A: Complaint link becomes NULL, message remains intact.

**Q: Can Head send messages to Admin?**  
A: Not directly. Head can only reply to Admin messages.

**Q: Are messages deleted automatically?**  
A: No, messages are permanent. Admins can manually clean old resolved messages.

**Q: Can I attach files?**  
A: Not yet. File attachments are a planned future feature.
